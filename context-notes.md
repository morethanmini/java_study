# README 작성 컨텍스트 노트

## 2026-06-20

- 이 저장소는 빌드형 Java 프로젝트가 아니라 브라우저에서 직접 여는 단일 HTML 자동채점 학습 파일 모음으로 판단했다.
- 루트에는 기존 README가 없었고, 챕터별 최신 HTML 파일은 각 챕터 폴더 바로 아래에 있다.
- `version/` 폴더는 이전 버전 보관용으로 README에 설명했다.
- 자동채점 문구와 코드 주석을 확인한 결과, 실제 Java 실행이 아니라 정규식 기반 문법 패턴 검증 방식이다. README에 이 한계를 명시했다.
- 문항 수는 최신 HTML 파일 기준으로 ch1 55문항, ch2 46문항, ch3 28문항, ch4 48문항으로 확인했다.
- 작업 전부터 `.DS_Store`와 ch1 폴더의 `.DS_Store`가 수정되어 있었다. 사용자 변경으로 보고 건드리지 않는다.
- `README.md`는 직접 출력해 제목, 표, 코드 블록, 경로 표기를 확인했다. 추적되지 않은 새 파일이라 `git diff --check`의 검사 대상에는 잡히지 않았다.
