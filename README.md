# Java Study

개인 학습용으로 만든 자바 문법 및 자료구조 자동채점 학습 파일입니다. 각 챕터는 브라우저에서 바로 열 수 있는 단일 HTML 파일로 구성되어 있으며, 코드를 입력하고 채점하면서 핵심 문법 패턴을 반복 연습할 수 있습니다.

## 학습 방식

1. 원하는 챕터 폴더로 이동합니다.
2. 최신 HTML 파일을 브라우저로 엽니다.
3. 문제의 TODO 영역에 코드를 입력합니다.
4. `채점` 버튼 또는 `Ctrl/Cmd + Enter`로 정답 패턴을 확인합니다.
5. 틀린 문제는 오답 노트로 다시 풀고, 필요한 문제는 즐겨찾기로 따로 모읍니다.

채점은 실제 자바 코드를 컴파일하거나 실행하는 방식이 아니라, 정규식으로 핵심 문법 패턴을 검증하는 방식입니다. 따라서 학습 흐름을 빠르게 점검하는 용도에 적합하며, 최종 코드는 IDE나 온라인 컴파일러에서 한 번 더 실행해보는 것을 권장합니다.

## 세부 학습 범위

### ch1. 자바 컬렉션 + 입출력 문법

- 배열과 컬렉션 선언 및 초기화.
- 생성, 조회, 수정, 삭제 같은 기본 CRUD.
- 검색, 정렬, 집합 연산.
- 오토박싱, 오버로딩 등 자주 헷갈리는 지점.
- Scanner, BufferedReader, StringBuilder 기반 입출력.
- 같은 패턴을 다른 데이터로 다시 푸는 변형 연습.

### ch2. 자바 기본 문법

- 캐스팅, `char`와 `int` 변환.
- 문자열과 숫자 간 변환 및 파싱.
- `equals`, `contains`, `indexOf` 등 String 비교와 검색.
- `substring`, `split`, `replace` 등 문자열 가공.
- `try-catch-finally`, `throws` 기반 예외처리.
- 같은 패턴을 다른 데이터로 다시 푸는 변형 연습.

### ch3. 자바 스택 + 큐

- Stack의 LIFO 구조, 괄호 검사, 후위표기법.
- Queue의 FIFO 구조와 BFS 방문 처리 패턴.
- Deque의 양쪽 끝 접근과 슬라이딩 윈도우.
- PriorityQueue, 힙, Comparator, 다익스트라 비교자 패턴.
- 같은 패턴을 다른 데이터로 다시 푸는 변형 연습.

### ch4. 자바 OOP + 람다 + Stream

- 클래스, 객체, 생성자, 필드, 메서드, `static`.
- 상속, `super`, 오버라이딩, 다형성.
- 인터페이스, `default` 메서드, `Comparable`.
- 캡슐화, getter/setter, `equals`, `toString`.
- 함수형 인터페이스, 람다, 메서드 레퍼런스.
- Stream API의 `filter`, `map`, `collect`, `reduce`.
- 같은 패턴을 다른 데이터로 다시 푸는 변형 연습.

## 기능 메모

- 브라우저만 있으면 실행할 수 있습니다.
- 입력한 답안과 진행 상황은 브라우저의 로컬 저장소에 저장됩니다.
- 진행 초기화 버튼으로 해당 챕터의 풀이 상태를 초기화할 수 있습니다.
- 오답 노트로 틀린 문제만 다시 풀 수 있습니다.
- 즐겨찾기로 다시 보고 싶은 문제를 따로 모을 수 있습니다.
- 정답 보기 기능으로 막힌 문제의 기준 풀이를 확인할 수 있습니다.

## 추천 학습 순서

1. ch2로 기본 문법과 문자열 처리 감각을 먼저 정리합니다.
2. ch1로 컬렉션과 입출력 패턴을 반복합니다.
3. ch3로 코딩테스트에 자주 나오는 자료구조 패턴을 익힙니다.
4. ch4로 객체지향, 람다, Stream 문법을 정리합니다.

이미 자바 기본 문법이 익숙하다면 ch1부터 시작해도 됩니다.

## 주의사항

- 자동채점은 핵심 문법 패턴을 확인하기 위한 보조 도구입니다.
- 정규식 기반 검증이라 논리적으로 맞는 다른 풀이가 FAIL 처리될 수 있습니다.
- 실제 실행 결과가 중요한 문제는 별도 Java 환경에서 컴파일 및 실행해 확인하세요.
- 브라우저 로컬 저장소를 삭제하거나 다른 브라우저를 사용하면 진행 상황이 이어지지 않을 수 있습니다.
